# MyUT NIM Otologin by Fikry

**Version:** 2.2

## Deskripsi
Otoinput NIM dan Otologin MyUT. Ekstensi ini membantu mahasiswa Universitas Terbuka untuk login secara otomatis ke laman MyUT tanpa perlu mengetik NIM secara manual berulang kali. Klik ikon ekstensi untuk mengatur dan menyimpan NIM Anda.

## Fitur
- **Penyimpanan NIM**: Simpan NIM Anda sekali, dan ekstensi akan mengingatnya.
- **Otologin**: Isi otomatis kolom NIM dan login dengan cepat.
- **Ringan & Aman**: Menggunakan penyimpanan lokal browser Anda.

## Instalasi
1. Unduh atau clone repository ini.
2. Buka Google Chrome dan navigasi ke `chrome://extensions/`.
3. Aktifkan "Developer mode" di pojok kanan atas.
4. Klik tombol "Load unpacked".
5. Pilih folder ekstensi ini (folder dimana `manifest.json` berada).

## Cara Penggunaan
1. Setelah terinstal, klik ikon ekstensi di toolbar browser.
2. Masukkan NIM Anda pada kolom yang tersedia dan klik 'Simpan'.
3. Buka halaman login [MyUT](https://myut.ut.ac.id/).
4. Ekstensi akan bekerja secara otomatis sesuai konfigurasi.
